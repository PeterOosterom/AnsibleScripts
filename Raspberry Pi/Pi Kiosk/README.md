# Pi Kios
Ansible script to transform a stock raspbian install into a full kiosk.

Run with:
ansible-playbook -e @vars.yml pikiosk.yml

https://github.com/azlux/log2ram/blob/master/ansible_playbook/install_log2ram.yml
